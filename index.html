ไฟล์รูปภาพสำรอง

ไม่จำเป็นต้องเลือกไฟล์ ถ่ายจากกล้องแล้วบันทึกได้ทันที ใช้ช่องนี้เฉพาะกรณีต้องการอัปโหลดรูปจากเครื่อง

การบันทึก

ถ่ายภาพจากกล้องแล้วกดบันทึกได้ทันที ไม่ต้องเลือกไฟล์ รูปจะตั้งชื่อตามเลขนักเรียน เช่น 19929.jpg
หลังบันทึก ระบบจะล้างรูปเดิมและเลื่อนไปนักเรียนคนถัดไป กล้องจะเปิดค้างไว้

Setting

กำหนด Google Sheet, Google Drive และพฤติกรรมการบันทึกไฟล์

ตั้งค่าฐานข้อมูลนักเรียน

ระบบนี้ตั้งค่าให้เชื่อมกับ Google Sheet ที่คุณส่งมาไว้แล้ว
คือรหัสกลาง URL ของ Google Sheet
จากลิงก์ใหม่ยังไม่มี gid จึงใช้ gid=0 หากไม่ใช่แท็บแรกให้เปลี่ยนเป็น gid ของแท็บข้อมูลนักเรียน

ตั้งค่าระบบบันทึกรูป

ตั้งค่า Drive และ Web App ถ้าต้องการอัปโหลดเข้า Google Drive จริง
ตัวอย่าง Folder ID คือข้อความหลัง /folders/ ใน URL ของ Google Drive
แทนที่ไฟล์เดิมเมื่อซ้ำชื่อ เมื่อเปิด ระบบจะตั้งใจให้ไฟล์ชื่อเดิมถูกแทนที่ หากใช้ Apps Script Web App สำหรับ Drive
ต้องใส่ URL ที่ลงท้ายด้วย /exec จาก Apps Script Web App ถ้าไม่ใส่ ระบบจะดาวน์โหลดลงเครื่องเท่านั้น
สำคัญ: ถ้าจะให้เว็บบน GitHub Pages อ่าน Google Sheet ได้โดยตรง ต้องตั้งค่าแชร์ชีตเป็น “ทุกคนที่มีลิงก์ดูได้” หากไม่ต้องการเปิดชีตสาธารณะ ต้องใช้ Apps Script Web App เป็นตัวกลาง

วิธีใช้งาน

โหมดถ่ายต่อเนื่องเร็ว: เปิดกล้องค้างไว้ กรอกเลขหรือใช้ปุ่มนักเรียนคนถัดไป → กด Enter เพื่อถ่ายภาพ → กด Ctrl+Enter เพื่อบันทึก ระบบจะเลื่อนไปคนถัดไปอัตโนมัติ
  1. เปิด Google Sheet ฐานข้อมูลนักเรียน แล้วตั้งค่าแชร์เป็น ทุกคนที่มีลิงก์ดูได้
  2. เข้าเมนู Setting ตรวจสอบ Sheet ID และ gid
  3. กด ทดสอบโหลดชีต หรือกลับหน้าแรกแล้วกด โหลดฐานข้อมูลใหม่
  4. หน้า ถ่ายภาพ กรอกเลขประจำตัว เช่น 19929 ระบบจะค้นหาข้อมูลนักเรียนจากชีต
  5. เปิดกล้องแล้วกดถ่ายภาพได้ทันที โดยไม่ต้องเลือกไฟล์ หรือเลือกไฟล์เฉพาะกรณีสำรอง
  6. กด บันทึกรูปจากกล้อง ระบบจะใช้ภาพที่ถ่ายล่าสุด ถ้ายังไม่ได้กดถ่ายภาพแต่กล้องเปิดอยู่ ระบบจะถ่ายให้อัตโนมัติก่อนบันทึก
